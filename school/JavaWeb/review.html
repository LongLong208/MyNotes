HTML

1. html全称Hyper Text Makeup Language
2. html分为单标签<> 和双标签<></>
3. 常见标签
换行 <br>
标题 <h1></h1>
段落 <p></p>
水平线标签 <hr>
文字设计标签 <font> </font>
    属性：size 大小；face 字体；color 颜色
文字风格标签
     <b></b> 加粗
     <u></u> 下划线
     <i></i> 斜体
     <sub></sub> 下标
     <sup></sup> 上标
     <ol></ol> 有序列表
     <ul></ul> 无序列表
4. 表格属性
    公共属性 align border width height
    特有属性
    bordercolor
    cellpadding 内容与边框宽度 （单元格内距，里面的里面）
    cellspacing 单元格与表格边框 （表格内距，最大的里面）
    colspan rowspan
5. 表单
    input标签属性type={text, password, radio, checkbox, reset, button, submit, image}
        ----image效果同submit，都会提交表单
    多行文字标签<textarea></textarea>
    
    下拉菜单标签<select></select>
        ---- <option></option> 选项
        ---- multiple属性：多选
        ---- size属性：显示项数
6. 注意     HTML不区分大小写

CSS

1. CSS全称Cascading Style Sheet，组成部分：对象、属性和值
2. CSS定义方法
     内联式：style属性
     内嵌式：<head></head>标签中写<style type="text/css"></style>
     链接式：<head></head>标签中写<link rel="stylesheet" type="text/css" href="xxx.css">
3. 选择器
     标记选择符：任何html元素（标签）
     类选择符：
        ---- .类名{}
        ---- 标记选择符.类名{}
     ID选择符 #id{}
     伪类：不同的状态，如 a:link a:visited a:hover a:active
4. 元素定位：流布局 绝对布局（z-index属性控制堆叠顺序）

z-index越高，越靠上
position 默认 static

JDBC

1. 获取连接对象
     Class.forName("com.mysql.jdbc.Driver");
     String url = "jdbc:mysql://localhost:3306/shiyan?characterEncoding=gb2312";
     Connection con = DriverManager.getConnection(url,"root","123");
2. 运行sql语句
     Statement stat = conn.createStatement();
     stat.executeQuery("sql查询语句");
     stat.executeUpdate("sql增删改语句");

     String sql = "sql语句";
     PreparedStatement ps = con.prepareStatement(sql);
     ps.setString(1, "xxx");      ps.executeQuery();

JSTL

<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
取session中的属性：<c:out value="变量名"></c:out>
移除session中的属性：<c:remove value="变量名" scope="session"></c:out>
在session中添加属性：<c:set value="变量名" var="值" scope="session"></c:set>
<c:if test="${条件}">内容</c:if>
<c:forEach var="元素名" items="集合名" begin="起始" end="结束" step="步长">代码段</c:forEach> escapeXml = true 把所有东西原样输出
